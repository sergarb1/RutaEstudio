Prompt detallado para generar una asignatura 
Eres un experto en planificación educativa. Genera un JSON válido para la
app "RutaEstudio" (una herramienta de mapas de conocimiento interactivos).
Quiero una asignatura completa sobre [TEMA].

# Estructura del JSON
{
  "subjects": [
    {
      "name": "Nombre de la asignatura",
      "description": "Descripción breve y motivadora (2-3 frases)",
      "concepts": [
        {
          "name": "Nombre del concepto",
          "description": "Explicación clara y sencilla del concepto (1-2 frases)",
          "weight": 5,
          "tags": ["etiqueta1", "etiqueta2"]
        }
      ],
      "relations": [
        { "from": "Nombre del concepto origen",
          "to": "Nombre del concepto destino",
          "type": "prerrequisito" }
      ]
    }
  ]
}

# Reglas estrictas
- Mínimo 10 conceptos, máximo 25
- weight: número entero del 1 al 10
  · 1-3: introductorio / de relleno
  · 4-6: importancia media / núcleo de la asignatura
  · 7-10: concepto fundamental / avanzado
- tags: array de etiquetas cortas (máximo 5 por concepto)
  · Usa etiquetas como "fundamentos", "avanzado", "práctico",
    "teoría", "ejercicios", "ejemplos", según corresponda
- En relations, el campo type debe ser uno de:
  · "prerrequisito" → A debe saberse antes que B (flecha sólida)
  · "pertenece" → A es subconcepto de B (flecha discontinua)
  · "relacionado" → A y B están relacionados (línea punteada)
  · "profundiza" → A profundiza / amplía B (flecha gruesa)
- Las relaciones deben formar un grafo acíclico (sin ciclos)
- Cada concepto debe aparecer al menos en una relación (entrante
  o saliente)
- Asegúrate de que todos los from/to coinciden exactamente con
  algún name en concepts
- No añadas campos extra ni comentarios
- Devuelve SOLO el JSON, sin markdown ni explicaciones

Puedes incluir resources opcionales en los conceptos:
"resources": [
  { "type": "video" | "articulo" | "libro" | "ejercicio",
    "title": "Título del recurso",
    "url": "https://..." }
]

Matemáticas — Álgebra lineal (2.º Bachillerato)

Eres un experto en planificación educativa. Genera un JSON válido para la
app "RutaEstudio" con la asignatura "Álgebra Lineal" de 2.º de Bachillerato.

Debe incluir 12-15 conceptos con pesos 1-10: matrices, determinantes,
sistemas de ecuaciones, rango, matriz inversa, combinación lineal,
dependencia lineal, espacio vectorial, subespacio vectorial, base y
dimensión, coordenadas, producto escalar, ortogonalidad, aplicaciones
lineales, núcleo e imagen, diagonalización.

Usa relaciones de prerrequisito (matrices → determinantes → sistemas…),
pertenece (subespacio → espacio vectorial), y profundiza (diagonalización
profundiza autovalores). Etiquetas: "fundamentos", "operaciones",
"teoría", "ejercicios", "selectividad", "avanzado".

Devuelve SOLO el JSON, sin explicaciones.

Bienvenido a RutaEstudio

Conecta conceptos, domina tu ruta. Estudia con propósito.

Organiza tus asignaturas como un grafo interactivo. Define conceptos, conéctalos con relaciones, evalúa tu dominio y obtén un plan de estudio personalizado.

Todo funciona 100% offline. Tus datos se guardan en el navegador.

Crea tu primera asignatura

Haz clic en "+ Nueva" o elige una plantilla educativa prediseñada con conceptos y relaciones ya creados. Tenemos plantillas de ESO, Bachillerato, FP Informática, FP Administrativo y transversales.

Los profesores pueden preparar la asignatura en 2 clics.

Añade conceptos

Cada concepto tiene nombre, descripción, peso/dificultad y etiquetas. Puedes añadirlos uno a uno, importarlos desde CSV o generarlos con IA.

También puedes editar el nombre y descripción con doble clic.

Conecta conceptos

Las relaciones definen cómo se conectan los conceptos: prerrequisito (A antes que B), pertenece (subconcepto), relacionado o profundiza.

Puedes crearlas desde el formulario o haciendo clic en el grafo.

Evalúa tu dominio

Puntúa cada concepto del 0 al 100% según tu nivel. Usa la evaluación uno a uno con las teclas 1 a 5, o la evaluación masiva con sliders.

Puedes hacer tantas evaluaciones como quieras y ver tu evolución.

Sigue tu plan de estudio

En la pestaña Resultados verás qué repasar (rojo), qué reforzar (amarillo) y qué dominas (verde). Usa los algoritmos BFS, DFS o el 🔥 Desbloqueador para priorizar.

También puedes usar el modo estudio con flashcards para repasar visualmente.
